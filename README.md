# ContextForge

An AI-native Next.js prototype for durable project knowledge and traceability.

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run build
```

## What’s included

- Core project-memory domain model
- Demo project graph and timeline
- Phase 1 planning docs in `context.md`, `plan.md`, and `todo.md`
